# MattRWX - Personal Website

## Simple & Dependency-Free

This website is built with pure **HTML, CSS, and Vanilla JavaScript**. 

It does **not** requires `npm`, `node_modules`, or any build tools to run.

## Getting Started

To view the website:

1. Open the `index.html` file in any web browser.

## Customization

You can edit the files directly:
- `index.html`: Update the structure and content.
- `style.css`: Modify the design and animations.
- `script.js`: Change the behavior and interactivity.
